🏐和phd: do not write yourself off

晚上讲了太多的话。

和导聊天，他和我大谈特谈半个小时phd的危害，好像听见我想去学界像是什么洪水猛兽。可是就是他问我要去学界还是业界的啊😂

他说这是一个需要非常非常非常谨慎的考虑过后才会做的选择，是在尝试了别的所有选择之后才做出的选择，因为有太多太多不确定性因素了，他说只要你还有别的路就不要走这条路，他见过许许多多比他聪明的多的人最终没有读博，或者甚至连博资考都没通过就quit了，我知道读书不会是想象的那么纯粹，但是我一开始的预想是对半开，但听他的描述，好像外界因素占80-90%的样子，你需要非常幸运，在正确的时间开始读正确的时间读完，每投一篇就是七八个月的等待，学历膨胀但是岗位缩招导致竞争变大，在政策变动之下人的个人能动性约等于0。读博会变穷，他在哥大吃不起寿司，只有做到AP教授才能实现财富自由，他现在赚得很多，但是他业界的同事赚的是他的十倍

他说，做到tenue会实现想要的生活，但是拿到tenue这个身份太难了，是90%的运气和10%的实力，如果可以，不要像他一样把所有的鸡蛋都放在一个篮子里。他有最曲折的学术道路，在工作上运气太好而在文章上得到了最坏的运气。这更多的是一个心理挑战而不是智力上的挑战，很多时候他的精神状态都非常不好，所以他并不担心我聪不聪明的问题，而非常担心在这之外的所有不确定性因素。

所以他很不情愿推荐我去读博，并且说如果不这样劝说他在道德上是非常不负责的。其实很好笑，因为phd这个东西存在在这里就是因为人类持续的不听劝啊，他说他会对每个想要读博的人都说这些，但是十个里面八个没听他的。所以最后的最后，这个决定只能出自于热爱，你没法解释，但足够的爱能不在乎一切的不确定因素，我们虽然研究理论，虽然作为理性的人做学术，但在追求学术这件事上，只有作为不理性的人才会做出这些决定，那这时候，也许这就是你适合的道路。

所以我们都决定再给我一些时间。去做研究，去写东西，像QMSS这样的项目，polisci的硕士并不值得读，因为大家都是phd显得这个硕士很没用，公共政策同理，如果你不感兴趣，并且这个学科上的训练并不能帮助phd的申请，纯粹为了工作而读也是他不想看见的。

他和我说了很多很多遍don’t write yourself off（“不要自我否定”意味着要停止认为自己是个失败者、无关紧要之人或者不具备成功的能力。避免过早地自我怀疑、丧失信心或放弃自己的潜力，它往往鼓励坚韧和自我信念，而非贬低自身的价值）我总说他太相信我了，我没跟他说我到底挣扎过多少，也不知道他理解了多少。老天，他说话甚至太滴水不漏了我挑不出一点不真诚的地方，或许他真的在说他的真心话，只是我不敢全部相信，觉得总有场面话夹杂在里面只是我真的分辨不出，这和别人太不一样了，我能很轻易地分辨出中介的客气话，然后提纯一下维持一个相对客观的判断，包括面对一般的赞美也能坦然接受然后抛之脑后，可是这个能力在导面前就自动失效，总是被轻易地感动总是过于轻飘地就想哭，一边不敢相信一边潜意识里早就信得死心塌地了

但是我还是太感谢有人和我说这些了。很多时候或许我就需要这样几段早就知道的说烂了的大道理，只是从一个更权威的前辈嘴里听见的感觉总是不一样的，很喜欢有人能提醒我现实的沉重但别轻易地屈服，其实我有些太依赖导，和其他所有的老师，但被认真地看着认真地说“你值得更好的”的感觉太好了，像上瘾一样，其实是对爱上瘾……我还是要说尤其是这些文科老师，接受了太好的训练所以随便聊天都能像艺术品一样，很多时候我不说，因为觉得要求别人鼓励甚至同情都有点超过了正常的礼貌界限，但是人总是自私的，于是人总是抱着一份隐隐的期待，其实理性来说不应该期待任何，但毫不意外的我每次都会得到120分的回复，他们总是能填满我的期待，然后溢出来…于是这一群人现在都成了和我关系最好的老师们（把自己养得很好！），所以其实我并不害怕申请季，因为我知道我身后有很多人的爱

有点离不开他们，虽然我知道最终的目标永远是自己爱自己，但接受他们的爱实在是感觉太好了，被划进保护范围内实在是太爽了，被主动安抚焦虑和鼓励实在是太爽了，这两天所有的不安就这样被填满，所以就这样吧，或许我也在因此慢慢变成更好的自己，变成这样的人类太迷人了。其实我这种人适合读书也不适合，太渴望遇上好老师就会超预期地蓬勃发展，要是遇上坏老师那就完蛋！但是西方教育体系的好处就是我可以自己选择接近的老师，而目前我的运气还挺好的。。。所以就这样成长吧

然后找带鱼聊了暑研偷偷多招人的小妙招，然后回去写了一大封邮件教程，就这样带导儿作奸犯科！作为新老师的开山大弟子的一大乐趣就是可以仗着自己老生的身份得知一些校内的潜规则然后带他干坏事😏坏处就是我是他第一个从头开始带毕业论文的学生，所以我不知道的事他也不知道😂这些都是小事，我的消息比他灵通，他这种雷霆日程的大忙人也不希望他操心这方面。前老板作为小领导还是太有钱了导致我以为工资都是学校发的，现在才知道是从个人funding里自掏腰包的，希望他有钱，没钱也可以，我一直觉得大家都穷穷的，但还是希望他有funding，这样我能赚点零花，他也不至于因为没法补偿我而愧疚。

🐟跟我说她今天打球不开心，因为一个是打得不好，觉得作为一起打球的水平最菜的一个很对不起学长辛辛苦苦擦地板救起来的球，因为听见他们蛐蛐别人的技术担心自己也被他们在背后蛐蛐，因为没照顾到dku的学姐把她一个人落下了所以很难过，因为觉得自己作为弱者的时候被保护很不开心好像有点自私…

她说了好久，我也说了好久，我说排球就是这样一个很坏的运动，如果大家都菜菜的也会玩得很开心，如果大家都强强的会打得更爽，如果作为场上唯一一个没跟上大家水平的人，感到隐形的边界是太正常的事情了，不管是被好心照顾还是被无意间忽视，其实哪一样都会很难受，但好像这不是任何一个人的错，所有人都会想要更好的游戏体验，但是菜更不是原罪，不接很多很多菜球也没法练成高手，好像是一个无法调和的矛盾一样。

但是你选择站在这个场上就已经比我勇敢。因为我不想承担这种难过，所以我选择了逃避，但你仍然喜欢去打不是吗，我上次和你打球，说你扣球稳定了不少，只是现在遇上了更厉害的人类，所以没必要为自己做不到的事情感到抱歉，也没有人理应承受这种情绪，但选择进步往往也意味着选择变得悲伤的可能。喜欢真的是很无解的事情，喜欢也真的是很不理性的行为，但只有这种冲动让我们能不顾流泪的风险仍然向前

我没法阻止你不内耗，因为我也内耗，但如果你想要单独提升技术，我很愿意陪你练，毕竟我也喜欢排球啊

Don’t write yourself off.

Don’t write myself off.

Don’t write ourselves off.

希望大家都不用内耗地成长为更强大的人类，但如果眼泪无法避免，那就大哭一场吧
